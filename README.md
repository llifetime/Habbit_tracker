# Habit Tracker (Atomic Habits)

Бэкенд-часть SPA веб-приложения для трекинга привычек, основанного на книге Джеймса Клира "Атомные привычки".

## 📋 Требования

- Docker 20.10+
- Docker Compose 2.20+
- Python 3.11+ (для локальной разработки)
- Git

## 🚀 Быстрый старт
